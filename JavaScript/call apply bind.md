# call apply bind

call、 apply、 bind 都是改变函数内部 this 指针的方法

|              | call()                                    | apply()                                  | bind()                                 |
| ------------ | ----------------------------------------- | ---------------------------------------- | -------------------------------------- |
| 传递参数方式 | 参数传递的方式为一个个传  arg1, arg2, ... | 参数传递方式为一个数组 [arg1, arg2, ...] | 参数传递方式为一个个传 arg1, arg2, ... |
| 调用函数方式 | 绑定并执行函数                            | 绑定并执行函数                           | 绑定但不执行函数                       |
