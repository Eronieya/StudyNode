# 函数节流和函数防抖

## 什么是函数节流和函数防抖

### 函数节流(throttle)

函数节流: 指定时间间隔内只会执行一次任务；

```js
function throttle(fn, time) {
    var timer
    return function() {
        if (timer) return;
        timmer = setTimeout( () => {
            clearTimeout(timer)
            timer = null
            fn()
        }, time)
    }
}
```



### 函数防抖(debounce)

函数防抖: 任务频繁触发的情况下，只有任务触发的间隔超过指定间隔的时候，任务才会执行。

```js
function debounce(fn, time) {
    var timer
    return function() {
        if(timer) {
            clearTimeout(timer)
            timer = null
        }
        
        timer = setTimeout( () => {
            clearTimeout(timer)
            
            timer = null
            fn()
        }, time)
    }
}
```
